# Data Processing Agreement (DPA - Lite)

This document outlines the high-level Data Processing terms for RiskLens Pilot Design Partners.

## 1. Scope of Processing
RiskLens acts as a **Data Processor**. We process security logs, IP addresses, email addresses, and incident metadata on behalf of the Customer (Data Controller) exclusively for the purpose of providing the Autonomous Incident Intelligence service.

## 2. Authorized Subprocessors
To deliver the service, RiskLens relies on the following infrastructure providers:
1. **Amazon Web Services (AWS)** / **Google Cloud Platform (GCP)**: Core hosting and database infrastructure (USA).
2. **OpenAI / Anthropic**: Large Language Model APIs used strictly for Root Cause Analysis (No training on customer data).
3. **Stripe**: Payment processing.

## 3. Data Protection
RiskLens agrees to maintain appropriate technical and organizational measures (TOMs) to ensure a level of security appropriate to the risk, including encryption in transit and at rest.

## 4. Breach Notification
In the event of a confirmed Data Breach impacting the Customer's data, RiskLens will notify the Customer without undue delay, and in any event within 48 hours.
